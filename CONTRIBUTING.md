# Contributing to straightmail

Thank you for your interest in contributing to straightmail! We welcome contributions from everyone.

## How to Contribute

### Reporting Bugs

- Use the GitHub issue tracker to report bugs.
- Provide a clear and concise description of the issue.
- Include steps to reproduce the problem.

### Suggesting Enhancements

- Open an issue to discuss your ideas for new features or improvements.

### Pull Requests

1. Fork the repository.
2. Create a new branch for your changes.
3. Make your changes and ensure they follow the project's coding style.
4. Write tests to verify your changes.
5. Submit a pull request with a clear description of what you've done.

## Code of Conduct

Please be respectful and professional in all your interactions with the project.
